## How To Use

<!-- Example: -->

Java JDK 8+ and [Leiningen](https://leiningen.org/) are needed if they don't exist locally.

```bash
# Clone this repository
$ git clone https://github.com/voromahery/image-uploader.git

# Install dependencies
$ npm install

# Run the app
$ npm start
```
